# 通用规则 - 卡牌信息

卡牌由名称、代价、类型、职业、属性、速度、力量、生命、耐久和规则文本等部分组成，这些共同定义了一张卡牌的特征。所有卡牌都具有名称、代价、类型、职业和属性。

带有职业奖励能力的卡牌具有水印图标，这些图标对应于该卡牌的职业，以及该职业奖励会为哪些英雄职业解锁。这些图标不影响游戏玩法。
