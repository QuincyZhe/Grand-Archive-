会忽略卡牌或效果上除首次提及外对多余的关键词与能力的任何提及。



### Agility[^1]&#x20;

1. 活力 N 是一种触发式能力，意指{在结束阶段开始时将 N 张卡牌从你的回忆区移回手牌。}。
2. 你可以选择将哪些卡牌移回你的手牌。
3. 如果你的回忆区中的卡牌数少于你能从敏捷中移回手牌的卡牌数，则尽可能地将卡牌移回手牌。
4. 卡牌上的多个活力会在结束阶段另开单独的触发。

### Ambush[^2]

1. 伏击是单位的静止式能力，意指{此单位即使其非防御物件，也可以对攻击者进行反击。}。
2. 单位必须保持备战状态，才能对攻击者进行反击。

### Brew[^3]

1. 酿造是一种静止式能力，允许使用替代代价启动卡牌，该替代代价涉及牺牲材料（通常通过采集召唤到战场上）。带有 Brew 关键词的卡牌会列出替代代价所需的物件。
2. 酿造意指{你可以牺牲列出的物件，而非支付此卡牌的封存代价}。
3. 使用酿造代价启动卡牌时，仍会受到额外启动代价修正的修改。修改封存代价的效果在酿造时会被忽略。

### [Champion Bonus](#user-content-fn-4)[^4]

1. 英雄奖励是一种制限式能力，其为卡牌定义了奖励效果或替换效果，只有当玩家的英雄的传承名称与该限制能力所指定的英雄名称相匹配时，该效果才会生效。
2. 英雄奖励在所有区域均有效。
3. 如果带有英雄奖励能力的卡牌所列出的英雄名称与玩家所控制的英雄名称不同，以至于如同无法满足职业奖励的条件那样，那么该奖励相关的关键词和效果在所有区域中均视为规则文本的一部分而被忽略。

### [Class Bonus](#user-content-fn-5)[^5]

1. 职业奖励是一种制限式能力，其为卡牌定义了奖励效果或替换效果，只有当玩家的英雄职业与该卡牌或物件的职业相匹配时，该效果才会生效。
2. 要解锁卡牌的职业奖励，玩家所控制的英雄必须与该卡牌上带有职业奖励的其中1个职业相匹配。
3. 职业奖励在所有区域均有效。
4. 如果带有职业奖励的卡牌与英雄的职业不同，以至于无法满足职业奖励的条件，那么该奖励相关的关键词和效果在所有区域中均视为规则文本的一部分而被忽略。

{% hint style="info" %}
例如，当《梅林，记忆窃贼》将玩家控制的《特里斯坦，幽灵追踪者》的墓场中的《火焰符文剑士》放逐时，在结算梅林的能力时，特里斯坦不会获得等级指示物。

{% endhint %}

### Cleave[^6]

1. 横扫是一种攻击、武器和单位的静止式能力，使它们能够攻击选定对手所控制的所有可攻击物件。
   1. 可攻击物件包括可攻占的领地。
2. 横扫攻击无法被拦截。
3. 带有横扫效果的攻击只涉及一个战斗步骤；对每个防守单位造成的伤害同时生效，并且在该战斗的反击步骤中为每个防守单位选择反击。
4. 在由横扫攻击发起的战斗步骤中，防守方的防守单位在攻击宣言时指定。在防守玩家控制下进入战场的任何单位都不会被指定为防守单位，也不会作为防守单位参与战斗。

{% hint style="info" %}
例如，被选中受到 Cleave 攻击的玩家在战斗的反击步骤开始时启动并结算《布兰奇，庇护圣徒》。布兰奇不会被视为防守单位，不能反击，也不会在伤害步骤中受到伤害。

![](<../.gitbook/assets/image (10).png>)![](<../.gitbook/assets/image (11).png>)
{% endhint %}

5. 带有横扫效果的攻击可能会被每个防守单位反击。
6. 每次伤害计算都是独立进行的，但伤害结算是同时发生的。
7. 带有横扫效果的攻击造成的伤害，用于该攻击的武器只会失去一个耐久指示物。
8. 横扫是不可重复的。

### Critical[^7] N

1. 暴击N 是一种静止式能力，意指{若此卡牌会造成战斗伤害，则改为造成双倍的伤害，除非 1 名对手弃置 N 张卡牌。}。
2. 任何对手即使其控制的单位未受到伤害，也可弃置 N 张卡牌。
3. 来自同一来源的多个暴击堆叠效果会叠加。例如，带有暴击 2 和暴击 3 的伙伴会让对手必须以任意顺序弃置 2 张和 3 张卡牌，否则将造成双倍的伤害。无论涉及多少个暴击效果，伤害只会改为双倍一次。

### [Divine Relic](#user-content-fn-8)[^8]

1. 神圣礼装是一种静止式能力，意指{你的素材库中只能有 1 张带有此关键词的卡牌。}。

### Efficiency[^9]

1. 效率是一种能改变代价的静止式卡牌或能力效果，意指{此卡牌启动时减少 LV 点代价}（LV 指你的英雄等级）。
2. 只有卡牌置于效果堆叠中，效率才会生效。

### [Element Bonus](#user-content-fn-10)[^10]

1. 属性奖励是一种制限式能力，其为卡牌定义了奖励效果或替换效果，只有当玩家的英雄属性与卡牌或物件的属性相匹配时，该效果才会生效。
2. 属性奖励在所有区域均有效。

### Empower[^11] N

1. 赋能 N 是一种静止式能力，意指{本回合你启动的下一张魔法卡牌将如同你都英雄等级提升 N 级一样启动并结算。}。
2. 赋能 N 会赋予“控制该卡牌或赋予该赋能效果”的玩家。其作为持续效果，会在已获得 赋能的玩家启动其下一张魔法卡牌时立即使用和生效。
   1. 赋能并非作为效果施加于被启动的卡牌上，但该卡牌可以检查玩家是否对其进行了 赋能。

### [Floating Memory](#user-content-fn-12)[^12]

1. 遥忆是一种静止式能力，意指{在支付回忆代价时，你可以将此卡牌从你都墓场放逐，以支付其中1点代价。}。
2. 在支付回忆代价时，使用遥忆支付的部分代价必须在确定作为支付回忆代价随机放逐的卡牌数量之前予以声明。
3. 遥忆是不可重复的。

### [Fast Activation](#user-content-fn-13)[^13]

1. 快速启动是一种静止式能力，意指{你可以以快速速度启动此卡牌。}。
2. 快速启动是不可重复的。

### Foster[^14]

1. 培育是一种触发式能力，意指{在你的回忆阶段开始时，如果此伙伴自你上一个回合结束以来未受到伤害，则其被指定成为培育状态。}。
2. 任何单位都可能被指定成为培育状态，但单位若无培育关键词则无法自行指定成为培育状态。
3. 培育是不可重复的。

### Hindered[^15]

1. 受阻是一种静止式能力，意指{此物件以待命状态进场。}。
2. 受阻是不可重复的。

### [Imbue ](#user-content-fn-16)[^16]N

1. 灌注是一种静止式能力，意指{你可以展示所有启动此卡牌被封存的卡牌。如果其中至少有N张卡牌的属性与该卡牌的属性相匹配，此卡牌被指定成为灌注状态。}。
2. 满足灌注能力的条件会使解决该卡牌或卡牌将要变成的物件被指定成为灌注状态。
3. 放置在回忆区并展示的卡牌并非从回忆区展示。那些只是在回忆区以正面被放置。这意味着其不会触发回忆区卡牌被展示的效果。
4. 成功启动灌注后，所展示的卡牌将被翻转至背面，并放置在效果堆叠中。
5. 如果卡牌上有多个灌注效果，那么只需满足最小的 N 值，该卡牌即可被指定成为灌注状态。

### Immortality[^17]

1. 不朽是一种静止式能力，意指{此物件不能从战场移至墓场。}。
   1. 不朽物件不会死亡；那些不会因生命值为0或更低而被送入墓场。
   2. 不朽物件不会被摧毁（包括通过牺牲）。
2. 如果玩家操控多个独有不朽物件，由于违反了独有规则，该玩家仍会基于状态效果被强制将其中一个物件物件置入其墓场。
3. 不朽是不可重复的。

### Inherited[^18] 效果

1. 继承效果是一种作用于英雄卡牌的静止式能力，意指{只要此卡牌是该英雄的传承卡牌之一，你的英雄就持有此能力。}。
2. 继承效果仅在带有该能力的卡牌处于传承中时生效。

### Intercept[^19]

1. 拦截是一种触发式能力，意指{每当你的英雄在该伙伴备战状态下受到攻击时，你可以将该攻击转向此伙伴。}。
   1. 要将攻击转向伙伴，该伙伴仍必须是有效的攻击目标。
2. 如果拥有拦截的伙伴不受玩家指令的影响，则拦截能力结算失败，该伙伴也无法进行 拦截。但响应仍成立，如果玩家有任何可能使伙伴受控制玩家指令影响的效果，拦截仍会触发。
3. 拦截是不可重复的。

### Kindle[^20] N

1. 点燃是一种静止式能力，意指{你启动此卡牌时，你可以从墓场中放逐至多N张火属性卡牌，每张放逐的卡牌为支付此卡牌的封存代价的 1 点。}。
2. 无论选择以何种方式支付代价，点燃不会改变卡牌的代价特征。

### 等级制限

1. 等级制限（以“Level N+ ”的气泡形式呈现）是一种制限式能力，用于检查玩家的英雄等级是否至少为 N 级。
2. 等级制限在所有区域均有效。

### Lineage[^21]（关键词）

1. 传承是一种用于升级英雄的静止制限式能力，意指{【卡牌名称】必须从名为 X 的前一级英雄升级而来}，其中 X 代表前一级英雄的名称。

{% hint style="info" %}
例如，《洛琳，流浪战士》属于洛琳卡名的英雄，满足升级为《洛琳，剑圣》的条件。

![](../.gitbook/assets/image.png)![](<../.gitbook/assets/image (1).png>)
{% endhint %}

[Lineage Release](#user-content-fn-22)[^22]

1. 传承解放是一种启动式能力，意指{通过将本卡牌从英雄的传承中放逐来启动此能力。}。
2. 卡牌上的传承解放能力不允许玩家通过放逐另一张拥有传承解放能力的卡牌来支付其所写能力代价；只有被放逐卡牌自身的能力效果会被置入效果堆叠等待结算。
3. 此能力仅当拥有传承解放的卡牌位于传承中时方可激活（即该卡牌非玩家的英雄单位）。

### Link[^23]

1. 链接是物件的一种静止式能力，可赋予被链接物件类型变化、数值或状态修正、额外能力。
2. 若物件通过效果堆叠结算后进入战场且其效果包含链接能力，则该物件将与指定目标物件形成链接（链接目标通常标注于“链接”关键词前）：
   1. 若持有链接能力的物件并非通过效果堆叠进入战场，其链接目标将在进场时直接选定，且不会触发任何目标指定过程。
   2. 若链接能力在启动或物化时无合规目标，则该能力失效。若持有链接能力的物件并非通过效果堆叠进入战场且无法选定链接目标，则该物件将被牺牲（详见第6、7条）。

{% hint style="info" %}
例如，英雄链接，伙伴链接。链接的目标物件即链接状态物件（可描述为“链接状态伙伴”“链接状态英雄”等）。
{% endhint %}

3. 具有链接能力的卡牌在置入效果堆叠时必须指定链接目标。若目标不合规，该能力失效。
4. 当持有链接能力的或链接状态的物件任一离开战场时，链接即断开。
5. 若因被链接状态物件离场导致链接被摧毁，则持有链接能力的物件将作为基于状态的行动被牺牲（玩家须在传递优先权或执行其他动作前完成牺牲）。
6. 同一物件可同时受任意数量的链接能力影响。
7. 仅当至少一方链接物件仍合规存在时，链接关系可维持。

### Memory[^24] N+

1. 回忆 N+ 是一种制限式能力，其为卡牌定义了奖励效果或替换效果，只有当玩家的回忆区中存在 N 张或更多卡牌时，该效果才会生效。

### [On Attack](#user-content-fn-25)[^25]

1. 攻击时是触发式能力的一种条件。
2. 单位上的攻击时，意指{当此单位攻击时。}。
3. 意图区和武器卡牌上的攻击时，意指{当此卡牌被用于攻击时。}。

### [On Death](#user-content-fn-26)[^26]

1. 死亡时是触发式能力的一种条件。
   1. 只有单位会死亡。非单位物体被摧毁时不视为死亡。
2. 单位仅当其直接进入墓场时才被视为死亡；若该单位被放逐（作为进入墓场的替代），则不会触发死亡时效果。
3. 死亡时效果在战场触发，且能力来源始终视为死亡的物件，而非最终进入墓场的卡牌本身。

### [On Enter](#user-content-fn-27)[^27]

1. 进场时是触发式能力的一种条件，意指{当本物件进入战场时。}。
2. 英雄上的进场时，其卡牌作为传承最上方的英雄卡牌进入传承时，视为卡牌已进入战场（它不是一个新物件）。

### [On Foster](#user-content-fn-28)[^28]

1. 培育时是触发式能力的一种条件，意指{当该单位被培育。}。

### [On Hit](#user-content-fn-29)[^29]

1. 命中时是触发式能力的一种条件。
2. 单位上的命中时，意指{当此单位对可攻击物件造成战斗伤害时。}。
   1. 造成伤害的形式是减少可攻占领地耐久指示物，也视为触发命中时效果。
3. 意图区和武器上的命中时，意指{当使用此卡牌进行的攻击或反击对可攻击物件造成战斗伤害时。}。

{% hint style="info" %}
例如，命中时效果可在任意单位受到战斗伤害时触发，通常通过攻击或反击。伙伴命中时仅在战斗伤害作用于伙伴单位时触发。英雄命中时仅在战斗伤害作用于英雄单位时触发。
{% endhint %}

4. 若命中时能力的效果描述中包含“该对手”或“该玩家”的表述，则此表述特指已命中物件的控制者。

### [On Kill](#user-content-fn-30)[^30]

1. 击杀时是触发式能力的一种条件，其判定时机为被攻击单位因攻击被击杀。
   1. 单位生命值为0以下时将作为基于状态的效果被击杀。
   2. 可攻占领地因其不被视为单位，耐久值为0以下将作为基于状态的效果被摧毁时不会触发击杀时效果。
2. 单位上的击杀时，意指{当单位因受到本单位的战斗伤害而被击杀时。}。
3. 意图区和武器上的击杀时，意指{当单位因受到本卡牌的攻击造成的战斗伤害而被击杀时。}。

### [On Leave](#user-content-fn-31)[^31]

1. 离场时是触发式能力的一种条件，意指{当本物件离开战场时。}。
2. 此效果在战场触发，且能力来源始终视为离场的物件，而非转移至其他区域的卡牌本身。

### Prepare[^32] N

1. 伺机 N 是一种可选的代价修正类的静止式能力，意指{启动【卡牌名称】时，你可以从己方英雄上移除 N 个伺机指示物作为额外支付。若如此做，【卡牌名称】将以伺机状态启动。}。
2. 若已支付伺机代价，则该卡牌变为“伺机状态”。

### Preserve[^33]/Preserved[^34]

1. 物件上的留存，意指{当【卡牌名称】被摧毁时，将其公开置入持有者素材牌库并标记为留存状态。当玩家将要物化时，可以作为物化的替代从素材牌库选择 1 张留存状态卡牌返回手牌。}。具有留存的物件被摧毁时，其对应卡牌从墓场转移至素材牌库并标记留存状态。
2. 非物件上的留存，意指{当【卡牌名称】结算完成且将被置入墓场时，改为将其公开置入持有者素材牌库并标记为留存状态。当玩家将要物化时，可以作为物化的替代从素材牌库选择 1 张留存状态卡牌翻回手牌。}。具有留存的非物件结算时产生替换效果，使其直接进入素材牌库而非墓场。
   1. 若具有留存的非物件卡牌结算时将被放逐而非置入墓场，则由控制者自由选择留存或放逐。
3. 所有留存状态的卡牌必须明置加入素材牌库，且保持明置直至转移至手牌、回忆区等非公开区域。
4. 留存状态是一种允许玩家在物化卡牌时选择取回的卡牌特性。
   1. 该操作不算作物化。
   2. 留存状态的卡牌在素材牌库必须保持公开。
   3. 玩家只要有物化机会即可作为物化的替代回收留存状态卡牌，无论其素材牌库中是否存在可物化卡牌。
      1. _“禁止双重替代”_。若允许“使用”卡牌，则只有选择物化操作才能够被替代，玩家将物化替代的同时不能启动卡牌。

### Pride[^35] N

1. 傲慢 N 是一种静止式能力，意指{若你的英雄等级未达到 N 级，此伙伴将不服从你。}。（参见服从规则。）
2. 若物件持有多重傲慢的场合，则只以最高 N 值作为服从判定标准。

### Ranged[^36] N

1. 远程 N 是单位的一种静止式能力，意指{当此单位处于遥远时，其攻击获得 +N 力量。}。
2. 多重远程的场合将叠加计算。

### Renewable[^37]

1. 可再生是一种礼装的静止式能力，意指{若此卡牌将从战场或意图区被放逐，则改为将其置入持有者的素材牌库。}。
2. 可再生不会在任何其他区域适用。
3. 可再生是不可重复的。

### Reservable[^38]

1. 可封存是一种物件的静止式能力，意指{支付封存代价时，你可横置此物件以支付1点该代价。}。
2. 此能力可以支付任何需要封存代价的代价，包括但不限于启动代价、追加或额外封存代价或可选类封存代价。
3. 可封存是不可重复的。

### Retort[^39] N

1. 反击 N 是一种静止式能力，意指{此伙伴反击期间，其获得 +N 力量。}。

### Spellshroud[^40]

1. 法术帷幕是一种物件的静止式能力，意指{磁卡不可被法术指定为目标。}。此外，法术被定义为{具有法术子类型的卡牌、效果或来源。}。
   1. 由法术物件生成的触发式能力本身继承法术子类型，亦无法指定具有法术帷幕的物件为目标。
2. 若物件在已被法术指定为目标的持续期间获得法术帷幕，该指定行为将被视为目标无效。当该目标是法术的唯一必要指定目标时，该法术直接失效。
3. 法术帷幕是不可重复的。

### Starcalling[^41]

1. 星谶是一种卡牌的静止式能力，意指{用预视查看此卡牌时，可支付其星谶代价将其启动。若如此做，需将预视查看的其余所有卡牌以任意顺序置入牌库下方。}。
   1. 此启动方式可无视常规启动时序规则。
2. 星谶是一种用于卡牌启动的替换类代价。
3. 当使用星谶代价启动卡牌时，玩家必须将预视查看的所有其他卡牌置入牌库下方，不能选择任何卡牌置入牌库上方。
4. 星谶代价仍受启动代价修正效果的影响。
5. 此能力仅在主要牌库生效。通过星谶启动的卡牌始终视为从主要牌库适用，其他区域无法启动此能力。
6. 通过星谶启动的卡牌将进入效果堆叠，但需等待当前所有结算中效果完全执行完毕后方可开始结算。

### Steadfast[^42]

1. 毅力是一种静止式能力，意指{此伙伴即使处于待命状态仍可反击，且反击时无需横置。}。
2. 毅力是不可重复的。

### Stealth[^43]

1. 潜行是属于单位的一种静止式能力，意指{此物件不可被攻击指定为目标，除非攻击者具有真视。}。
2. 若物件在获得潜行前已对攻击进行反击宣言，仍可正常对原反击目标执行伤害结算。
3. 若物件在被攻击指定的持续期间获得潜行，该攻击指定将被视为目标无效，除非攻击物件持有真视，否则攻击直接失效。同时被攻击物件停止成为防御物件，攻击物件停止成为攻击物件。
4. 潜行是不可重复的。

### Taunt[^44]

1. 嘲讽是一种静止式能力。
2. 当指定攻击目标时，玩家必须优先指定备战状态且具有嘲讽的物件，而非其他可攻击目标。
3. 若战场上同时存在多个嘲讽物件，攻击玩家可自由选择其中任意物件攻击。
4. 嘲讽是不可重复的。持有多个嘲讽的物件不会叠加优先级，与仅具有单一嘲讽的物件同等。

### [True Sight](#user-content-fn-45)[^45]

1. 真视是一种静止式能力。
   1. 单位上的真视，意指{此单位的攻击可指定具有潜行的物件为目标。}。
   2. 武器上的真视，意指{通过此武器的攻击可指定具有潜行的物件为目标。}。
2. 真视是不可重复的。

### Unblockable[^46]

1. 贯穿是一种静止式能力，意指{此单位的攻击不可被拦截且无视嘲讽。}。
2. 来自贯穿单位的攻击的目标仍可被改变。
3. 贯穿是不可重复的。

### Vigor[^1]

1. 活力是一种单位的触发式能力，意指{此单位在你的结束阶段开始时转变为备战状态。}。
2. 当单位具有多重活力标识时，每个标识都会在结束阶段产生独立的触发。
